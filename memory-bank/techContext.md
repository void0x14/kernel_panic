# Tech Context

## Technologies Used
- **Frontend:** Vanilla HTML/Javascript + Three.js.
- **Styling:** Vanilla CSS (Modern CSS properties, Glassmorphism, Sophisticated Color Palettes).
- **Assets:** Procedurally generated geometry, potentially 360-degree placeholders or generated textures.

## Development Setup
- Local Development: **Vite** (via `npm run dev`).
- Language: ES6 Modules (Native).
- OS: Linux (CachyOS/Void kernel).

## Technical Constraints
- No Purple/Violet colors.
- No Hacker Green/Black theme.
- Focus on performance (fastest implementation without excessive neon/complex post-processing).
- Multi-input accessibility.
