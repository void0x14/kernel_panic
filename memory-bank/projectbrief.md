# Project Brief - Segmented Tunnel

## Core Requirements
- A 3D interactive "Segmented Tunnel" (Structural Tunnel) visualization.
- The tunnel is an infinite cylinder composed of discrete rings representing time units (e.g., Year/Month).
- Interactive panels (cards) are arranged along the inner walls of each ring.
- Content types: Images, text, videos, and audio.
- Clicking a panel triggers an immersive 360-degree content display.

## Goals
- Create a clean, sci-fi museum-like experience.
- Ensure high performance and smooth navigation ("rail system").
- Build a modular and accessible control system (Keyboard + Mouse).
- Establish a flexible data structure for future dynamic expansion.
